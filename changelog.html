My Game Of Life - Change Log

changelog.txt

Version 1.0.3 (2008-01-11)
  • Automatically saves user settings:
    • Cell behaviour
    • World size and colors
    • Window size and position
  • Allow cut/copy/paste and clear selected cells.
  • Improved cycles per second indicator.
  • Added zoom level indicator.
  • Made compatible with MacOS X 10.4.
Version 1.0.2 (2007-12-31)
  • Customizable world size (up to 2046 x 1022, almost 2MB cells!).
  • Added scroll bars to navigate world view.
  • Added zoom slider on main screen.
  • Added cell cycle counter.
Version 1.0.1 (2007-12-28)
  • Added process indicator.
  • Added alive cell counter.
  • Added cycles per second indicator.
Version 1.0 (2007-12-27)
  • Initial version.
